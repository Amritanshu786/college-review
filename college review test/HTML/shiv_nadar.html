MODE OF ADMISSION

  • For BTech, BSc, BA and BMS programmes, admissions are done based on the candidate’s performance in Class 12 along with Shiv Nadar University Scholastic Aptitude Test (SNUSAT) and Academic Proficiency Test (APT).
  • Candidates applying through JEE Main scores need to have scored more than 120 marks to be eligible for BTech programme.
